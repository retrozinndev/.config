# `retrozinndev/.config`
My personal dotfiles for [colorshell](https://github.com/retrozinndev/colorshell).

Includes configs like `fastfetch`, `kitty` and `Hyprland` + `colorshell` configs.
Also, this config implements support for [XDG Autostart](https://wiki.archlinux.org/title/XDG_Autostart) even if you're not using 
[UWSM](https://wiki.archlinux.org/title/Universal_Wayland_Session_Manager)! (you need to manually enable it inside `autostart.lua`)

## Icon Theme
I personally use the [MoreWaita](https://github.com/somepaulo/MoreWaita) icon theme,
it has a lot of symbolic icons, making the system look a lot nicer.

## Apps
- [`valent`](https://valent.andyholmes.ca), desktop phone integration (notifications, calls, fs mounting, and a lot more!)
- [`zen-browser`](https://zen-browser.app), beautiful firefox-based web browser
- [`neovim`](https://neovim.io), text editor I use for... editing text files ([my config](https://github.com/retrozinndev/nvim.lua))
- [`amberol`](https://gitlab.gnome.org/World/amberol), the music player (it's pretty, simple and fast)
- [`vesktop`](https://vesktop.vencord.dev), the best discord client
